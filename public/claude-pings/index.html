Setup

Use it with Claude Code.

Three steps to hear your ping after every task.

  1. 1
    Download a ping from above.
  2. 2
    Save it somewhere permanent, like:
    C:\Users\YourName\.claude\my_ping.wav
  3. 3
    Add a hook to ~/.claude/settings.json:
    // Add inside settings.json { "hooks": { "Stop": [ { "matcher": "", "hooks": [ { "type": "command", "command": "powershell.exe -NoProfile -Command \"(New-Object Media.SoundPlayer 'C:\\Users\\YourName\\.claude\\my_ping.wav').PlaySync()\"" } ] } ] } }

That's it. Every time Claude Code finishes a task, your custom ping plays.
